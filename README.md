# YouTube UI Clone

A simple front-end clone of the YouTube homepage built using **HTML and CSS**.
This project includes a navigation bar, sidebar menu, video grid layout, and a shorts section—styled to closely resemble YouTube’s interface.

## Features

* YouTube-style navbar with search, mic, and sign-in button
* Scrollable sidebar with menu options
* Video cards with thumbnails, duration tags, and channel info
* Shorts section with vertical video cards
* Clean and modern UI using pure HTML & CSS

## Tech Stack

* **HTML5**
* **CSS3**
